# Projeto---Analise-de-dados_-Power-BI

## 📝 Descrição
Este projeto tem como objetivo realizar uma análise exploratória de dados (EDA) da Olist, focando em métricas de performance financeira, comportamento de consumo por região e métodos de pagamento preferidos pelos clientes.

## 📊 Principais Insights
* **Volume de Vendas:** Identificamos que o estado de São Paulo é o principal motor de receita.
* **Comportamento:** O cartão de crédito domina as transações em todos os estados.
* **Categorias:** A categoria Automotiva teve o maior desempenho financeiro.

## 🛠 Tecnologias Utilizadas
* **Power BI:** Para a criação dos dashboards e modelagem de dados.
* **DAX:** Para criação de medidas calculadas (ex: Receita Total).
* **Git/GitHub:** Para controle de versão.

## 🚀 Como acessar

2. Abra o arquivo no seu **Power BI Desktop**.
