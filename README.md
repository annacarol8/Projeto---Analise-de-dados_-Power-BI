# Projeto: Analise de dados - Power BI

## Análise de Vendas e Performance – E-commerce Olist

#📝 Sobre o Projeto

Este repositório contém a análise de dados completa da plataforma Olist, desenvolvida ao longo das etapas AC1, AC2, AC3 e AC4. O objetivo é transformar dados brutos de e-commerce em insights acionáveis para a gestão de vendas, logística e satisfação do cliente.


🔹 AC1: Análise de Vendas Regional e Mercado
* Foco: Identificação dos principais polos de faturamento e comportamento inicial.
* Destaques: Distribuição geográfica das vendas, ranking de categorias mais vendidas e métodos de pagamento preferidos.
  
🔹 AC2: Eficiência Logística e Prazos
* Foco: Performance de entrega e cumprimento de prazos.
* Destaques: Comparativo entre data de entrega e análise de gargalos logísticos por estado.
  
🔹 AC3: Qualidade e Satisfação do Cliente
* Foco: Experiência do utilizador e feedback pós-compra.
* Destaques: Análise de pontuação (Reviews), relação entre atrasos e notas baixas, e principais motivos de insatisfação.
  
🔹 AC4: Análise de Faturamento e Evolução Mensal
* Foco: Saúde financeira e análise temporal.
* Destaques: Comparativo de vendas ano a ano, evolução mensal do faturamento e indicadores de crescimento.
  
#🛠️ Tecnologias e Metodologia
* Ferramenta Principal: Power BI Desktop.
* Linguagem DAX: Criação de medidas para KPIs de crescimento e inteligência de tempo (ex: Receita Total).
* Power Query: Limpeza, transformação e modelagem de dados.
* Gestão do Projeto: Organização de tarefas via Trello.

#📌 Fonte dos dados: 
Os dados utilizados foram extraidos do Dataset público da Olist no Kaggle, Segue: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce?resource=download&select=olist_order_items_dataset.csv
